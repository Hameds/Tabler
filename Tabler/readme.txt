﻿--- What is Tabler?
Tabler (https://tabler.github.io/) is free premium and Open Source dashboard template with responsive and high quality UI and this package contains Tabler files and sample layout for ASP.NET MVC projects

--- How to get started
after installing Tabler nuget package you can see _TablerSampleLayout.cshtml which is sample layout based on Tabler empty page and create your customized layout.
for more information about Tabler seed docs (https://tabler.github.io/tabler/docs/index.html)

--- Contribute
Tabler source code is available on github (https://github.com/Hameds/Tabler)
